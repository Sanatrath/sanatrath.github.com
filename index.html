PeerSpot / AR /

PeerSpot is an Augmented Reality iPhone companion app to help Georgia Tech students find the fastest mode of intra-campus transport in real time.

With the user in a rush, the idea was to show only the most meaningful information to decide from after meticulously calculating & comparing bus locations, occupancies, cycle availability and buffer times in the backend.

Awaaz / AAC /

Awaaz is an Augmentative and Alternative Communication iPad app for 'technophilic' older adults with speech or language impairments.

The goal of the system was to help users frame coherent meaningful sentences effectively and efficiently from simple contextual hints and with minimum stress on working memory. The app relies on visual modality and also accomodates the possibility of language changes.

Usability Principles: Visibility / Consistency / Error Correction / Auto-Suggestion / Feedback / Social Input

Finance App //

The Financial Need Analysis module is a self-aided tool to help families understand and plan for their insurance, education and retirement based on existing financial assets, liabilities and goals.

The challenge at hand was to make the process of input playful and emotive, unlike the existing web app. Further, user should be able to visualize valuable output with the least effort. The concepts explored various metaphoric design ideas and gestures for I/O including touch, click, Drag & Drop, Pinch and Spread. User Testing was carried out to evaluate the designs.

Deal Daaku / web /

As an exercise for travel sites with fluid experiences, DealDaaku explored a number of web based interaction concepts related to searching among the best travel deals available.

The first concept was a widget based interface which rearranges itself uniquely to different users according to the currently active and most used features. The second concept enabled the user find and book the cheapest flights directly from a monthly calendar. The final concept, bubble, was envisioned as a more fluid interface where the user could filter, or rather wade, through and save from a group of deal bubbles.
